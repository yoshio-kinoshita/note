# チームファイブテンの青と黒
チームファイブテンの青と黒の違いっていまいちよく分かっておりません。
公式サイトにも詳しい説明はなさそうです。で、周りから聞いて話をまとめてみます。

# 初代チームファイブテン青
もう売ってないと思いますが、初代チームファイブテンは相当固かったらしいです。
トーのラバーは剛性を保つためでフックを使うのも難しかったらしいです。
履きづらさはピカイチ？

# 二代目チームファイブテン青
トーフックの弱点や硬さをマイルドになりました。剛性は落ちましたが、オールマイティーな靴になったのかな？
ただ、履きづらさは相変わらず。
